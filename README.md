# 산책16기 카드뉴스 공유 사이트

이 폴더는 Cloudflare Pages와 GitHub 저장소로 자동 배포되는 카드뉴스 공유 사이트입니다.

- 메인 파일: `index.html`
- 이미지 경로: `cardnews/YYYY-MM-DD/01.png` 형식
- 검색 노출 방지: `noindex` 메타 태그, `robots.txt`, `X-Robots-Tag` 헤더 설정 포함

## 새 카드뉴스 추가 규칙

1. 강의 날짜로 폴더를 만듭니다. 예: `cardnews/2026-06-12/`
2. 카드뉴스 이미지를 `01.png`, `02.png`처럼 두 자리 숫자 이름으로 저장합니다.
3. `index.html`의 날짜 목록과 이미지 경로를 새 날짜 폴더 기준으로 추가합니다. 카드뉴스가 14장이면 14장까지 추가하면 됩니다.
4. GitHub에 올리면 Cloudflare Pages가 자동으로 새 사이트를 배포합니다.

## 현재 등록된 카드뉴스

- `2026-06-04` 비견과 겁재 / 정재와 편재
- `2026-06-02` 내 주변인에게 나는 어떤 사람일까?
